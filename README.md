# Markdown Editor

ブラウザ上で動作する、リアルタイムプレビュー対応のシンプルなMarkdownエディタです。

## 機能

* **リアルタイムプレビュー**: 左側のエディタに入力すると、即座に右側にHTML形式で表示されます。
* **GitHub Flavored Markdown (GFM) 対応**: 見出し、リスト、テーブル、タスクリスト、打ち消し線など、一般的なマークダウン記法をサポートしています。
* **Mermaid.js 対応**: ```mermaid コードブロックを使用することで、ダイアグラムやフローチャートを描画できます。
* **LaTeX 数式対応**: KaTeXを使用し、`$` で囲んでインライン数式を、`$$` で囲んでブロック数式を表示できます。
* **ファイル読み込み**: ローカルの `.md` ファイルを読み込んで編集できます。
* **画像挿入**: ローカルの画像ファイル（PNG, JPEG, SVGなど）を選択し、Markdownに埋め込むことができます。
* **上書き保存**: 読み込んだファイル名でそのまま保存します。
* **名前を付けて保存**: 新しいファイル名でダウンロード保存できます。
* **PDF出力**: 「PDF出力」ボタンから、ブラウザの印刷機能を使用してプレビュー内容をPDFとして保存できます。
* **編集内容保護**: 編集中にページを離れようとすると警告が表示され、誤ってデータを失うのを防ぎます。

## 対応しているマークダウン記法

本エディタはGitHub Flavored Markdown (GFM) に準拠しており、以下の記法をサポートしています：

* **見出し**: `#`, `##`, `###` など
* **リスト**: 箇条書き (`*`, `-`, `+`) および順序付きリスト (`1.`, `2.`)
* **タスクリスト**: `- [ ]`, `- [x]`
* **テーブル**: `| ヘッダー | ヘッダー |` 形式の表
* **強調・装飾**: 太字 (`**text**`), 斜体 (`*text*`), 打ち消し線 (`~~text~~`)
* **コード**: インラインコード (`` `code` ``) およびコードブロック (```言語名 ... ```)
* **Mermaidダイアグラム**: ```mermaid ... ```
* **数式**: インライン (`$E=mc^2$`), ブロック (`$$\frac{a}{b}$$`)
* **引用**: `> 引用文`
* **リンク・画像**: `[タイトル](url)`, `![画像タイトル](url)`

## 実行方法

1. `index.html` ファイルをブラウザ（Chrome, Edge, Firefoxなど）で開きます。
2. 左側のテキストエリアにMarkdownを入力してください。
3. 必要に応じて「開く」ボタンで既存のファイルを読み込むか、「画像挿入」ボタンで画像を挿入し、「保存」ボタンで編集内容を保存してください。

## 技術スタック

* **HTML/CSS/JavaScript**: 標準的なWeb技術で構成。
* **Tailwind CSS**: UIのスタイリング。
* **marked.js**: MarkdownからHTMLへの変換。
* **Mermaid.js**: Markdown内でのダイアグラム描画。
* **KaTeX**: Markdown内での数式描画。
